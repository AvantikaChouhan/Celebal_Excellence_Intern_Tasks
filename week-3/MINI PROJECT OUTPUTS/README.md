# 📈 Mini Project: Customer Sales Insights

## 🎯 Objective

The objective of this mini project is to analyze customer sales data from the **Superstore Dataset** using SQL queries and generate meaningful business insights.

This project focuses on identifying customer purchasing behavior, sales performance, and customer ranking using advanced SQL concepts.

---

# 🗂️ Dataset Information

* 📁 Dataset: Sample - Superstore
* 💻 Database: MySQL
* 📄 File Format: CSV
* 🧾 Main Tables Used:

  * customers
  * orders
  * products

---

# 🧠 SQL Concepts Used

The following SQL concepts were used in this mini project:

* 🔹 JOIN Operations
* 🔹 Aggregate Functions (`SUM`, `MAX`, `COUNT`)
* 🔹 GROUP BY
* 🔹 HAVING Clause
* 🔹 Subqueries
* 🔹 Common Table Expressions (CTEs)
* 🔹 Window Functions

---

# 📊 Business Insights Generated

The project analyzes customer sales performance to identify:

* ⭐ Top-performing customers
* 📉 Lowest-performing customers
* 🛒 Customers with only one order
* 💰 Customers with above-average sales
* 🏆 Highest order value for each customer

---

# ✅ Tasks Performed

### 1️⃣ Top 5 Customers

Identified customers with the highest total sales.

### 2️⃣ Bottom 5 Customers

Identified customers with the lowest total sales.

### 3️⃣ Customers with Only One Order

Analyzed customer order frequency to find one-time customers.

### 4️⃣ Above-Average Sales Customers

Identified customers whose total sales are greater than the average customer sales.

### 5️⃣ Highest Order Value Per Customer

Calculated the maximum order value for every customer.

# 🎉 Conclusion

This mini project demonstrates how SQL can be used for customer sales analysis and business intelligence reporting.

By using SQL queries and analytical techniques, meaningful insights were generated from the Superstore dataset regarding:

* 📊 Customer behavior
* 💵 Sales trends
* 🏅 Customer performance
* 🛍️ Order analysis
