# 📊 Superstore Sales Analysis using SQL

## 🎯 Objective

The objective of this project is to analyze the **Superstore Sales Dataset** using advanced SQL concepts such as:

* 🔹 Subqueries
* 🔹 Common Table Expressions (CTEs)
* 🔹 JOIN Operations
* 🔹 Window Functions

This project focuses on generating customer sales insights and understanding sales performance through SQL queries.

---

# 🗂️ Dataset Used

* 📁 Dataset Name: **Sample - Superstore**
* 📄 File Format: CSV
* 💻 Database Used: MySQL
* 🧾 Imported Table: `superstore_raw`

---

# 🛠️ Database Setup

## 📌 Step 1: Database Creation

A database named `superstore` was created and the CSV dataset was imported into a raw table called `superstore_raw`.

---

## 📌 Step 2: Table Creation

Three normalized tables were created from the raw dataset:

### 👥 customers

Stores customer-related details:

* Customer_ID
* Customer_Name
* Segment

---

### 📦 products

Stores product-related information:

* Product_ID
* Product_Name
* Category
* Sub_Category

---

### 🧾 orders

Stores order and sales details:

* Order_ID
* Order_Date
* Ship_Date
* Ship_Mode
* Customer_ID
* Product_ID
* Sales
* Quantity
* Discount
* Profit

---

# 🧠 SQL Concepts Used

## 🔹 Subqueries

Used for analyzing sales and customer order patterns.

---

## 🔹 Common Table Expressions (CTEs)

Used for calculating customer sales summaries and advanced filtering.

---

## 🔹 Window Functions

Used for customer ranking and row numbering operations.

---

## 🔹 JOIN Operations

Used to combine customer, order, and product information for analysis and reporting.
