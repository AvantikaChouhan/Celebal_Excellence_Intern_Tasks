# 🚀 Celebal_Excellence_Intern_Tasks

## 📖 Overview

This repository contains weekly assignments and projects completed during the Celebal Technologies Data Engineering Internship Program.

The work covers data cleaning, SQL database design, advanced SQL analytics, cloud-based data pipelines, and big data processing using Apache Spark.

The assignments demonstrate practical skills in:

- Data Cleaning and Preprocessing
- SQL Database Design and Querying
- Advanced SQL Analytics
- Azure Cloud Services
- Azure Data Factory (ADF)
- Data Pipeline Development
- Big Data Processing using Apache Spark
- Data Transformation and Aggregation
